# discord-bot-mu
Bot discord criado com objetivo de adicionar/recuperar informações dos boss no jogo mu online para que os jogadores consigam se organizar.

#### Tecnologias utilizadas:
- typescript
- discordjs
- firestore
- rabbitmq
